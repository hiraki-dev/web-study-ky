リソースタグ

sourceタグ

親要素がpictureタグ、audioタグ、videoタグの場合にメディアリソースを指定する際に使用する。
親要素によって指定できる属性が変わる。
参考

  • 親要素がpictureタグの場合、srcset属性でリソースパスを指定する。
  • 親要素がaudioタグ、videoタグの場合、srcタグでリソースパスを指定する。
  • 共通属性として、type属性でリソースの種類を指定する。
  • 親要素がpictureタグの場合、以下の属性を指定できる。
    • sizes属性で画像サイズを指定
    • media属性でメディアクエリ（レスポンシブ幅等）を指定
    • width属性、height属性で縦幅、横幅を指定

pictureタグ

ページ内に表示する画像の選択肢をグルーピングする。
要素として、ひとつ以上のimgタグ、resourceタグで構成する。
参考

imgタグ

ページ内に画像リソースを表示する。

  • src属性でリソースパスを指定する。
  • alt属性で画像読み込み失敗時の代替テキストを指定する。
  • その他の属性はリンク先を参照すること。

iframeタグ

ページ内にインラインフレームを作成する。
指定したURLのリソースを読み込み、ページ内に表示する。
リンク先のHTMLソースやメディアなどの表示に利用する。

  • src属性でリソースパスを指定する。
  • srcdoc属性でフレーム内のドキュメントを指定する。（srcより優先される）
  • sandbox属性でインラインフレームのセキュリティ（DL許可設定等）を指定する。
  • allow属性で権限を指定する。
  • その他、属性の詳細はリンク先を参照すること。

objectタグ

外部リソースを表示する。

  • data属性でリソースパスを指定する。
  • type属性でリソースの種類を指定する。
  • form属性でformタグと関連付けを行う。
  • その他、属性の詳細はリンク先を参照すること。

video/audioタグ

メディアリソースを表示する。
各属性を用いて、表示幅、自動再生の有無などをコントロールする。
videoタグは動画メディアを、audioタグは音声メディアを表示する。（属性は共通）

  • リソースはsrc属性かsourceタグを用いて指定する。
  • autoplay属性を指定するとページ読み込み完了時にメディアを再生する。（非指定時は再生しない）
  • preload属性でメディア表示の最適化を行う。
  • controls属性を指定するとユーザコントロールタブを提供する。
  • poster属性でメディア再生前のポスターフレームを表示する。
  • trackタグでvideoタグの埋め込みメディアに字幕を指定できる。
  • crossorigin属性でCORSを制御する。
  • その他、詳細は以下のリンクを参考にすること。
    • video - HTML Living Standard リファレンス
    • video - 動画埋め込み要素 - mdn web docs
    • HTML5でvideoタグを使って動画を埋め込む方法を簡単解説！

map/areaタグ

イメージマップを作成する。
指定したイメージ画像にクリック可能領域とクリック時のアクションの指定を行う。
mapタグでイメージマップのフレームを作成し、area属性で動作設定を行う。

  • mapタグ
    • name属性は必須で、マッピング対象のimgタグに指定したusemap属性の値を指定する。
    • idを指定する場合、nameと同じ値を指定する。
  • areaタグ
    • href属性で領域クリック時のリンク先を指定する。
    • shape属性で領域の座標指定方法を指定する。
      • shape="rect"（長方形）の場合、「x1, y1, x2, y2」で値を指定
      • shape="circle"（円形）の場合、「x, y, radius」で値を指定
      • shape="poly"（多角形）の場合、「x1, y1, x2, y2, .., xn, yn」で値の指定
      • 上記値のcoords属性に付与
      • sharp="default"の場合、画像全体を座標として指定（coords属性は指定不可）
    • alt属性で画像読み込み失敗時の代替テキストを指定する。
    • その他、属性の詳細はリンク先を参照すること。

画像をクリックすると解説ページへ移動。

説明を省略するタグ

  • embedタグ（objectタグで代替可能）
  • MathML（数式を表示するマークアップ言語）
