型の種類

JavaScriptの型はメソッド、プロパティを持つオブジェクト型 、メソッド、プロパティを持たないプリミティブ型に分けられる。

  • 下記の表はプリミティブ型を記載。
  • オブジェクト型はリンクを参照。
型 説明
number 整数、不動小数点数、10進数、16進数等の数値型
string 文字列型
boolean 真偽値
function 関数型
object オブジェクト型
Symbol シンボル型
true, false 真偽値
null ヌル値
undefind 未定義
NaN 数値ではない（Not a Namber）
Infinity 無限値
