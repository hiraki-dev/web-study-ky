配列（リスト）

リスト状に並べられたデータの集合体のことで、配列（Array）、リストなどと呼ばれる。
配列は以下のように[ ]で囲って作成する。 

let list = [ 'index1', 'index2', ... ];
let list = [];

二次元配列

配列の多次元化は以下のように記述する。

let two_dimension_array = [
  [ 1, 2, 3 ],
  [ 'a', 'b' ],
  [ 
    [ 1, 2], 
    [ 'c', 'd' ],
  ],
];

let temp = two_dimension_array[0][2]; // 2

配列メソッド

length

配列のindex数を取得する。

let list = [ 1, 2, 3 ];
list.length;  // 3

unshift

配列を取り出す。

push

配列を追加する。

連想配列（ディクショナリ）

KeyとValueをセットに持つ値のことで、マップ、ディクショナリなどと呼ばれる。
JavaScriptではMapオブジェクトとして定義されている。
オブジェクトと違い下記のメリットがあり、プログラムの中で単なる連想配列を扱いたい場合にマップを宣言すればいい。

  • キー値に文字列やシンボル以外も扱える。
  • リスト数をsizeメソッドで簡単に得られる。
  • for of構文で簡単にループ処理出来る。
//マップのインスタンス
let map = new Map(); 
//値の追加
map.set('testNam', 100);
//値の取得（100）
map.get('testNam');

集合（セット）

値が重複せず、追加した順番が保持されるリストのこと。
JavaScriptではSetオブジェクトとして定義されている。
