概要

JavaScriptでは全ての型をオブジェクトとして扱うため、それぞれの型には組み込みのメソッドがサポートされている。
文字列や数値などの値を変数にセットした際、内部的にそれぞれに対応したオブジェクトのインスタンスが生成される。

数値

数値はNamberオブジェクトとしてインスタンスされる。
数値処理関連のメソッドを使用できる他、直接呼び出すことも可能。 

// 指定された値が数値か判定
Number.isNaN(100);

文字列

文字列はStringオブジェクトとしてインスタンスされる。
文字列処理関連のメソッドを使用できる他、直接呼び出すことも可能。 

// 文字列を引数に指定した文字列で区切り、リストとして返す
String.split( 文字列 )

// 文字列を引数に指定した文字列で検索し、hitした場合最初にhitした文字列の開始位置の値を返す
String.indexOf( 文字列, 検索開始位置 );

// 置換対象に指定した値を置換処理（コールバック関数指定可）に指定した文字列に置き換える
String.replace( 置換対象, 置換後文字列 );

// 文字列を変数にセットすると文字列型のオブジェクトが生成される
let userId = "ｈｏｇｅ　ｈｕｇａ";

// 空白削除
userId = userId.Replace(/\s+/g, "");
// 全角入力値を半角に変換
userId = userId.Replace(/[0-9]|[,]/g,
  function(str) {
    return String.fromCharCode(str.charCodeAt(0) - 0xFEE0);
  }
);

配列

数値はArrayオブジェクトとしてインスタンスされる。
配列処理関連のメソッドを使用できる他、直接呼び出すことも可能。 

// 配列オブジェクトの作成
let list = [1, 2, 3];

// 配列のアイテム総数を取得
list.length();    // 3 

// リストを引数に指定した文字で結合し、文字列として返す
list.join(”-”);   // "1-2-3"

// リストの要素を引数に指定したコールバック関数で検査し、検査結果がtrueの値のみのリストを生成して返す
list.filter((list) => {
  for (let i of list) {
    if (i % 2 === 0) {
      return true;
    }
  };
});
