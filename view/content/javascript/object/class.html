クラスの定義

// クラス
class クラス名 {
  // コンストラクタ
  constructor( ) {
    ...
  }

  // メソッド（functionは不要）
  メソッド名( ) {
    ...
  }

  // スタティックメソッド（インスタンスしなくても呼び出せる）
  static メソッド名 {
    ...
  }

  // ゲッターとセッター
  get type( ) {
    ...
  }

  set type( ) {
    ...
  }
}

// クラスのインスタンス
let test = new クラス名();
// メソッド呼び出し
test.メソッド名;
// setter
test.type = 値;
// getter
console.log(test.type);

prototype

プロトタイプ。
定義したクラスに変数やメソッドを追加できる。 

let test = new クラス名()
// プロパティ（クラス変数）を追加
test.prototype.変数名 = 値;
// メソッドを追加
test.prototype.メソッド名 = () => { ... };

extends

継承。 継承元をスーパークラス（親クラス、または基底クラス）、継承先をサブクラス（子クラス、または派生クラス）と呼ぶ。 

class 親クラス {
  ...
}

// 継承はextendsを指定する
class 子クラス extends 親クラス {
  ...
}

super

クラスを継承したとき、子クラスからコンストラクタやメソッドを呼び出す際に利用する。
変数にクラスを格納することで匿名クラスを定義出来る。 

let test = class {
  constructor () {
    this.type = 1 ;
  }
};

console.log(test.type);
