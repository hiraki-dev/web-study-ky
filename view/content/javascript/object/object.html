概要

JavaScriptでは、数値、文字列、配列、真偽値などの型が内部的にはオブジェクトとして扱われ、様々なメソッドやプロパティがサポートされている。

オブジェクトの宣言と参照

オブジェクトは以下のように宣言する。

// 通常の定義方法で、{ }で囲んだ範囲がオブジェクトとなる
let obj = {
  key : value
};
  • オブジェクトはkeyとvalueをセットに持つ値の集合体であり、{ }で宣言する。
  • オブジェクトに定義された値はメンバーという。
  • メンバーは.（ドット）または[ ]で指定して参照する。
    • 基本的に簡易的なドット記法（.）が好まれる
    • メンバー名の指定に変数を用いる場合、ブラケット記法（[ ]）を使用する

以下はオブジェクトの定義と参照の例である。

// オブジェクトの宣言
let user = {
  name : "山田",
  age : 30
};

// オブジェクトの参照
console.log(user.name);       // 山田

let userName = "山田";
console.log(user[usernName]); // 山田

他の言語でいう連想配列（ディクショナリ）に似た形でjavaScriptではオブジェクトを実装できる。
ディクショナリとの違いは以下である。

  • キーをプロパティとして扱う。
  • 値に関数を定義出来る。
  • プロパティに定義出来るのは文字列とシンボルのみ。

通常のディクショナリは Mapオブジェクトとしてサポートされている。

オブジェクトメンバーの追加

オブジェクトに新たにメンバーを追加する場合、ドットまたはブラケットで追加したいメンバーのkeyを指定し、任意の値を代入する。 

let obj = {
  name: tanaka,
  age; 30
};

obj.profile = "プロフィールを追加";
// { name : tanaka, age : 30, profile: "プロフィールを追加" }
console.log(obj);
