コードブロックハイライトの導入

目次

  1. 概要
  2. 導入方法

概要

タグの開始とタグの終了をハイライトで分かりやすく表示してくれる機能。
Blockman - Highlight Nested Code Blocksを拡張機能として導入するとHTML、CSS、JavaScriptなどの言語をハイライトしてくれる。

導入方法

Blockmanの拡張機能をインストールする。

導入するとコードブロックハイライトが表示されるが、デフォルトだと表示に癖がある。

以下は設定例である。
設定 -> 設定（JSON）を開くでsettings.jsonを開き、以下の設定内容をペーストすると冒頭のハイライト表示になる。
※波括弧の中身をコピペすること 

{
   /*
    * blockmanの設定内容
    */
    // ブロックの表示幅を最大化
    "blockman.n04Sub04RightSideBaseOfBlocks": "Rightmost Edge Of File Content",
    // ブロックフォーカス時のカラー
    "blockman.n17CustomColorOfFocusedBlock": "none",
    // ブロックフォーカス時のボーダーカラー
    "blockman.n18CustomColorOfFocusedBlockBorder": "cyan",
    // 第二階層以降のブロックボーダーカラー
    // "blockman.n19CustomColorOfBlockBorder": "blue",
    // 第一階層のブロックボーダーカラー
    // "blockman.n20CustomColorOfDepth0Border": "yellow",
    // ブロックボーダーの角丸
    "blockman.n21BorderRadius": 0,
    // ブロックボーダーの太さ
    "blockman.n34A02BorderThicknessOfFocusedBlock": 0.85,
    // 非フォーカス時のボーダーの太さ
    "blockman.n34A01BorderThicknessOfNonFocusedBlock": 0,
    // コードブロックのレンダリングスピード
    "blockman.n28TimeToWaitBeforeRerenderAfterLastChangeEvent": 0.5,
    // ブロックカラー（階層順）
    "blockman.n05CustomColorOfDepth0": "none",
    "blockman.n06CustomColorOfDepth1": "none",
    "blockman.n07CustomColorOfDepth2": "none",
    "blockman.n08CustomColorOfDepth3": "none",
    "blockman.n09CustomColorOfDepth4": "none",
    "blockman.n10CustomColorOfDepth5": "none",
    "blockman.n11CustomColorOfDepth6": "none",
    "blockman.n12CustomColorOfDepth7": "none",
    "blockman.n13CustomColorOfDepth8": "none",
    "blockman.n14CustomColorOfDepth9": "none",
    "blockman.n15CustomColorOfDepth10": "none",
}

Blockman導入時に推奨構成のアナウンスが表示される。（2024/04/09時点）
VSCode側の設定に慣れる意味も込めて、以下の内容もペーストして挙動を確かめておくと良い。 

// settings.json (User/Global config, not Workspace config)
