@ルール（アットルール）

CSSでは @アットルール名 値; という書式で定義された@ルールを使用してCSSの動作を定義する。
以下は@ルールの使用例である。 

/*  
  外部CSSファイルをインポートする
  @importは最上部に定義する必要がある
*/
@import url('./asset/example.css');

/* 文字コードを定義 */
@charset "utf-8";

以下に主な@ルールの種類を記載する。
詳細はリンク先を参照すること。

@ルール名 動作
@import 外部CSSファイルを読み込む。
@charset 文字コードを指定する。
@media 指定したメディアクエリの条件を満たした場合、設定を適用する。
@page ページ印刷時の設定を適用する。
@color-profile CSSのカラープロファイルを定義する。

@ルールのネスト

以下の@ルールはネストして定義出来る。
@ルールのネストはスタイル適用時に &セレクタ 指定時と同等のルールが適用される。

  • @media
  • @supports
  • @layer
  • @scope
  • @container

以下は@ルールのネスト例である。
詳細はリンク先を参照すること。 

/* 例 */
.example {
  ...
  @media (orientation: landscape) {
    ...
  }
}

/* 上記と以下は同じ挙動である */
.example {
  ...
}

@media (orientation: landscape) {
  .example {
    ...
  }
}
