CSS（Cascading Style Sheets）とは

HTMLで記述した要素の装飾スタイルを指定する言語のことで、WebページのデザインはCSSで記述を行う。
CSSではデザインを指定したいHTMLの要素（セレクタ）を指定し、どのような装飾（スタイリング）を行うかを記述する。 

/* 以下はCSSの記述例 */
@charset "utf-8";

body {
  background-color: rgb(222, 233, 244);
}

CSSの書き方

セレクタ {
  プロパティ:プロパティ値;
}
  • セレクタでHTMLのタグや属性を指定する。
  • セレクタを指定後、{ } でスタイルルールを適用するための入れ子（ネスト）を作成する。
  • プロパティで要素の装飾スタイルとその値を指定する。
  • プロパティと値のセットを宣言という。

宣言を適用するセレクタは以下のように複数指定可能。

p,
li,
h1 {
  color: red;
}

CSSの適用方式

CSSには HTMLタグ埋め込み、HTMLページ内埋め込み、 CSSファイル読み込みの三種類の適用方式があり、 これらの方式によってデザイン適用の優先順位が変わる。

適用優先順位は以下の通りである。

  1. HTMLタグ埋め込み
  2. HTMLページ内埋め込み
  3. CSSファイル読み込み

HTMLタグ埋め込み

<!-- 例 -->
<p style="color:red;">あいうえお</p>

HTMLページ内埋め込み

styleタグに直接CSSを記述し、ページ内に適用する。

<style>
div {
  background:red;
}
</style>

CSSファイル読み込み

基本的にCSSは別ファイルに分けて管理し、linkタグで読み込みを行いページに適用する。

<!-- common.cssというcssファイルをページ内に適用する例 -->
<head>
  <link rel=“stylesheet” type=“text/css” href=“common.css”>
</head>

<!-- styleタグを用いたインポートも可能 -->
<style type="text/css">
  @import “common.css”;
</style>
