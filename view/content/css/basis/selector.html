要素セレクタ

要素やタグを指定してスタイルを指定する。

p {
  color: red;
}

ユニバーサルセレクタ（全称セレクタ）

全ての要素を指定してスタイルを適用する。

/* セレクタには*（アスタリスク）を指定する */
* {
   color: red;
}

クラスセレクタ

セレクタにclass属性を指定することで、そのclass名を持つタグにスタイルを適用する。 

<div class=“test1”>
    <p class=“test2”>
        ...
    </p>
</div>
.test1 {
    color: red;
}

/* 親要素を指定すると指定した親要素を持つクラスにスタイルが適用される */
p.test2 {
  color: blue;
}

IDセレクタ

セレクタにid属性を指定することで、そのid名を持つタグにスタイルを適用する。 

<div id=“test1”>
    <p id=“test2”>
        ...
    </p>
</div>
#test1 {
    color: red;
}

p#test2 {
    color: blue;
}

子孫セレクタ

セレクタを空白で連結して指定することで、先頭のセレクタを親に持つ子セレクタを指定出来る。

  • 子孫セレクタの場合、間に別のセレクタがある場合でも適用される。
  • セレクタはタグ名、クラス名が指定出来る。
div p {
    ...
}
<div>
    <ul>
        <p>...</p><!-- div配下のpタグに適用される -->
    </ul><!-- 間にulタグが存在しても適用される -->
</div>

子セレクタ

セレクタを > で連結して指定することで、先頭のセレクタを親に持つ子セレクタを指定出来る。
子孫セレクタとは違い、間に別のセレクタがある場合は適用されない。 

div > p {
    ...
}
<div>
    <p>...</p><!-- divセレクタ直下のpタグに適用 -->
</div>

<div>
    <ul>
        <p>...</p><!-- 左記のように直下でない場合適用されない -->
    </ul>
</div>

隣接セレクタ

セレクタを + で連結して指定することで、先頭のセレクタに隣接するセレクタを指定出来る。
間に別のセレクタがある場合は適用されない。 

div + p {
    ...
}
<p>pタグ</p>
<div>divタグ</div>
<p>divタグに隣接したpタグ</p><!-- divタグに隣接したpタグに適用 -->
<p>pタグに隣接したpタグ</p>

関節セレクタ

セレクタを ~ で連結して指定することで、先頭のセレクタの配下にあるセレクタを指定出来る。
先頭セレクタの配下の外には適用されない。 

div ~ p {
    ...
}
<div>
    <h1>...</h1>
    <p>...</p><!-- 適用 -->
    <h2>...</h2>
    <h3>〜</h3>
    <p>...</p><!-- 適用 -->
</div>
<p>...</p><!-- 非適用 -->

属性セレクタ

特定のプロパティや値を持つタグにスタイルを指定出来る。

  • タグ名[プロパティ名]と記述する。
  • プロパティ名は複数指定出来る。

属性セレクタの基本

/* 例 */
img[src][alt] {
    ...
}

/* 値の指定 */
img[src=“hoge.jpg”] {
    ...
}
<img src="">
<img src="" alt=""><!-- 適用1 -->
<img src="hoge.jpg"><!-- 適用2 -->

応用として、以下のような指定方法もある。（参考）

OR条件

<-- OR条件適用(スペース区切りで判断) -->
<style>
p[class~="bar"] {
  color : green;
}
</style>
 
<p class="foo bar">適用される</p>
<p class="foobar foo">適用されない（対象外）</p>
<p class="bar">適用される</p> 

<-- OR条件適用（ハイフン区切りで判断） -->
<style type="text/css">
p[lang|="en"] {
  background-color: #6EB3E5;
}
</style>
 
<p lang="en">適用される</p>
<p lang="ja">適用されない（対象外）</p>
<p lang="en-US">適用される</p>

前方一致

<-- 前方一致 -->
<style type="text/css">
a[href^="http://"] {
  color: orange;
}
</style>
 
<p><a href="https://www.webdesignleaves.com/">適用される</a></p>
<p><a href="#id">適用されない（対象外）</a></p>
<p><a href="https://www.google.com/">適用されない（対象外）</a></p>

後方一致

<-- 後方一致 -->
<style type="text/css">
a[href$=".pdf"]{
  background-color: yellow;
}
</style>
 
<p><a href="manual.doc">適用されない（対象外）</a></p>
<p><a href="sample.pdf">適用される</a></p>
<p><a href="sample.xls">適用されない（対象外）</a></p>

部分一致

<-- 部分一致 -->
<style type="text/css">
img[src*="/thumb/"] {
  border: 2px solid #ccc;
}
</style>
 
<p><img src="images/small/sky.jpg"></p><!-- 対象外 -->
<p><img src="images/thumb/sea.jpg"></p><!-- 適用される -->
<p><img src="images/large/wood.jpg"></p><!-- 対象外 -->
