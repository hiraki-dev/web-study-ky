ブロックボックス/インラインボックス

CSSはブロックボックスとインラインボックス の二つのブロックで構成されている。
主に以下の違いがある。

  • ブロックボックス
    • 改行が挿入される。
    • widthプロパティ、heightプロパティが適用される。
    • paddingプロパティ、marginプロパティ 、borderプロパティが適用される。
    • 要素が追加された場合、要素を囲うようにインライン（内側方向）にサイズが大きくなる。
      • widthプロパティが指定されている場合はインラインが固定となる。
  • インラインボックス
    • 改行が挿入されない。
    • widthプロパティ、heightプロパティが適用されない。
    • paddingプロパティ、marginプロパティ 、borderプロパティは以下のように適用される。
      • 上下方向には適用されるが、他のインラインボックスには影響しない。
      • 左右方向にはブロックボックスと同様に適用される。

HTMLタグのボックスについて

divタグとspanタグの明確な違いがどちらのボックスに属しているかである。
デフォルトではdivタグはブロックボックスに属し、spanタグはインラインボックスに属している。

  • ブロックボックス
    • セクションタグやグルーピングタグなどの構造を定義するタグが属する。
    • divタグ、pタグ、ulタグなど
  • インラインボックス
    • テキストタグなどの装飾を行うタグが属する。
    • aタグ、bタグなど

displayプロパティによるボックスの明示

displayプロパティを用いることでタグのボックスを変更できる。

/* divタグをインラインボックスに変更 */
div {
  display: inline;
}

/* spanタグをブロックボックスに変更 */
span {
  display: block;
}

/* inline-blockを指定すると、margin、border、pattingの値が有効なインラインボックスとなる */
b {
  display: inline-block;
}

ボックスモデル

CSSのボックスは以下の要素で構成されている。

ブロックボックスの構成要素
ボックス名 説明
コンテンツボックス（Content） コンテンツの表示領域。
inline-size、block-size、width、heightなどのプロパティでサイズを変更する。
パティングボックス（Padding） コンテンツの周囲の領域。
padding関連プロパティでサイズを変更する。
ボーダーボックス（Border） コンテンツとパティングを囲う枠線。
border関連プロパティでサイズを指定する。
マージンボックス（Margin） ボーダーの外側の余白領域。
marginプロパティでサイズを決定する。

デフォルトのボックス設定（通常フロー）

HTMLタグがどのボックスに属するか、マージン、ボーダー、パティングの値などのデフォルト値はブラウザ毎に規定されている。
CSSプロパティに値を指定することで通常フローから値が変更される。

ブラウザの開発者ツール

各種ブラウザの開発者ツールを用いることで要素ごとに割り当てられたスタイル宣言、ボックスモデル領域などを確認できる。

標準ボックスモデル/代替ボックスモデル

CSSのボックスモデルには標準ボックスと代替ボックスが用意されている。

  • box-sizingプロパティで指定可能。
  • 値をcontent-boxに指定した場合、標準ボックスとなる。
    • width、heightはコンテンツボックスのサイズとして計算される
    • border、pattingはコンテンツの外側に加算される
  • 値をborder-boxに指定した場合、代替ボックスとなる。
    • width、heightはマージンを除いたボックス全体のサイズとして計算される
    • border、pattingはコンテンツの内側に加算される

各ボックス毎のプロパティ

マージンボックス

マージン領域は以下のプロパティで上下左右の幅を指定できる。
リンク先で動作検証可能。

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

ボーダーボックス

  • 上下左右の幅は以下のプロパティで指定する。
    • border-top
    • border-right
    • border-bottom
    • border-left
  • 幅、スタイル、色などは以下のプロパティで指定する。
    • border-width
    • border-style
    • border-color

より詳細なプロパティ、動作などはリンク先を参照すること。

パティングボックス

パティング領域は以下のプロパティで上下左右の幅を指定できる。
リンク先で動作検証可能。

  • patting-top
  • patting-right
  • patting-bottom
  • patting-left
  • 指定できる値は整数のみ

マージンの相殺

要素が隣接したとき、各要素ごとのマージンに指定された値により以下の挙動となる。

  • 整数同士の場合は加算した合計値となる。
  • 一方に負数が指定されている場合、一方の整数値から差し引かれる。
  • 負数同士の場合は負数の合計値となる。
  • 上下の要素が隣接したときは以下の挙動となる。
    • 上側の要素のmargin-bottomの値が優先される
    • 下側の要素のmargin-topの値が上側のmargin-bottomを超過した場合、超過分加算される
    • margin-bottomに負数を指定した場合、下側のmargin-topから差し引かれる
    • margin-topに負数を指定した場合、上側のmargin-bottomから差し引かれる

詳細はリンク先を参照すること。
