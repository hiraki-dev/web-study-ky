CSSのネスト

CSSの定義は { } 内に記述するが、 { } の中に更に { } を作成して定義を行える。
これを ネスト という。
以下のように、ネストを効果的に用いることでスマートで可読性の高い記述が出来る。 

/* ネストの例 */
div.example {
  /* divタグ内に適用するスタイル */
  ...
  p {
    /* 該当divタグを親要素に持つpタグに適用するスタイル */ 
    ...
  }
}

/* 上記と以下は同じ挙動である */
div.example {
  ...
}

div.example p {
  ...
}

&セレクタを用いたネスト

&セレクタを使用することで以下のネスト記述が出来る。

  • 半角スペースで接続できない疑似クラス/疑似要素などのネスト化
  • 親要素をネストで追加

疑似クラス/疑似要素などのネスト

疑似クラス/疑似要素の前に & を付与することでネスト出来る。

/* 親セレクタがホバーされた場合の例 */
div.example {
  ...
  &:hover {
    /* 親要素がホバーされた際に適用 */
    ...
  }
}

/* 上記と以下は同じ挙動である */
div.example {
  ...
}

div.example:hover {
  ...
}

親要素を追加するネスト

セレクタの後に & を付与することで、親要素をネストで追加できる。 

/* 例 */
.supplement {
  ...
  .example & {
    /*
      class="example"が指定されたdivタグ内の、
      class="supplement"が指定された要素にスタイルを適用
    */
    ...
  }
}

/* 上記と以下は同じ挙動である */
.supplement {
  ...
}

.example .supplement {
  ...
}
