疑似クラス/疑似要素

疑似クラス/疑似要素を使用することで指定セレクタの特定の要素、特定の状況でスタイルを適用できる。

  • 擬似クラス
    • 指定セレクタが特定の状態のときにデザインを適用出来るセレクタ。
    • セレクタの後ろに :擬似クラス名 と指定することで適用できる。
  • 疑似要素
    • 指定セレクタの一部に対してデザインを適用出来るセレクタ。
    • セレクタの後ろに ::擬似要素名 と指定することで適用できる。
  • 詳細な種類は以下を参照すること。
    • スタイルシートリファレンス（目的別） - セレクタ
    • CSSリファレンス(セレクタ)

疑似クラスの適用例

link/visited

<style>
  a:link {
    ...
  }

  a:visited {
    ...
  }
</style>

<!-- :linkはaタグに指定したリンクが未訪問の場合にスタイルを適用する -->
<a href="...">テキスト</a>
<!-- :visitedはリンク先に遷移済の場合に適用する -->
<a href="...">テキスト</a>

hover/active

<style>
  .class1:hover {
    ...
  }

  .class2:active {
    ...
  }
</style>

<!-- :hoverはタグに対してマウスが置かれた時にスタイルを適用する -->
<dev class="class1">...</div>
<!-- :activeはタグがアクティブ状態の時にスタイルを適用する -->
<dev class="class2">...</div>

疑似要素の適用例

first-letter/first-line

<style>
  .class1::first-letter {
    ...
  }

  .class2::first-line {
    ...
  }
</style>

<!-- ::first-letterはタグ内の一文字目にスタイルを適用する -->
<p class=“class1”>テキスト</p>
<!-- ::first-lineはタグ内の一行目にスタイルを適用する -->
<p class=“class2”>テキスト</p>

before/after

<style>
  /* 以下の疑似クラスはcontentプロパティの指定が必須 */
  .class1::before {
    content: "挿入する文字列や画像等"
  }

  .class2::after {
    content: "挿入する文字列や画像等"
  }
</style>

<!-- pタグの前にcontentの内容が追加される -->
<p class=“class1”>テキスト</p>
<!-- pタグの後にcontentの内容が追加される -->
<p class=“class2”>テキスト</p>
