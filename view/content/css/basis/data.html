データ型について

CSSではプロパティに指定された値を適切に解釈するため、値を判別するためのフォーマット（データ型） が組み込まれている。
プロパティごとに指定可能なデータ型が指定されており、適切な値を指定しないとレイアウトを適用できない。

  • プロパティに必要な値がテキストの場合は文字情報の羅列、url関数を用いたURL値など
  • プロパティに必要な値が色の場合はカラーコード(#FFFFFFなど)やrgb関数を用いたRGB値など
  • プロパティに必要な値が大きさや位置の場合はピクセルなどの単位表記付きの整数、負数など
  • 全てのプロパティに指定できる値はユニバーサルプロパティ値と呼ばれる。（unset等）

データ型ごとの単位

ここではデータ型ごとに、よく使われる単位を纏めておく。
とほほのCSS入門より引用

color型

色名、カラーコード値、rgb関数などで指定する。

/* 色名 */
p {
  color: red;
}

/* カラーコード */
p {
  color: #FFFFFF;
}

/* rgb関数 */
p {
  color: rgb(255, 255, 255);
}

text型

文字列、url関数などで指定する。

/* 例 */
body {
  background-image: url(...);
}

length型

length型の単位には相対長（相対的な長さ）と絶対長（絶対的な長さ）がある。

  • 相対長はフォントサイズやビューポートなどの長さと相対した長さで、他の要素のサイズによって変動する。
  • 絶対長は固定の長さで、他の要素のサイズに左右されない。

文字

単位 種別 説明
em 相対 親要素にのフォントサイズが指定されている場合、その値を基に割合を算出する。
例として、親要素が12pxであれば1emは12px、3emは36px。
親要素に値が指定されていない場合は16pxとなる。
rem 相対 ルート要素（htmlタグ）に指定されたフォントサイズを基に割合を算出する。
デフォルトは16px。
ex 相対 半角文字「x」の高さ(x-height)を基に割合を算出する。
多くのブラウザでは0.5emとして扱われる。
rex 相対 ルート要素（htmlタグ）に指定された半角文字「x」の高さ(x-height)を基に割合を算出する。
ch 相対 半角文字「0」の横幅を基に割合を算出する。
rch 相対 ルート要素（htmlタグ）に指定された半角文字「0」の横幅を基に割合を算出する。

大きさ/長さ

単位 種別 説明
% 相対 パーセント。
親要素のサイズを基に割合を算出する。
cm 絶対 センチメートル（1cm=10mm）。
mm 絶対 ミリメートル（10mm=1cm）
Q 絶対 1/4ミリメートル（1/4mm）。
in 絶対 インチ（1in=2.54cm）
pt 絶対 ポイント（1pt=1/72in）
pc 絶対 パイカ（1pc=12pt）
px 絶対 ピクセル（1px=1/96in）

ビューポート

単位 種別 説明
vw 相対 ビューポートの横幅（viewport's width）。
横1920pxの画面の場合、100vw = 1920px。
vh 相対 ビューポートの高さ（viewport's height）。
縦1000pxの画面の場合、100vh = 1000px。
vmin 相対 ビューポートの横幅と高さの小さな方の値を参照する。
（viewport's mininum length）
vmax 相対 ビューポートの横幅と高さを比較して大きな方の値を参照する。
（viewport's maxinum length）

CSS4以降、より詳細なビューポート単位として以下が追加されている。

単位 種別 説明
svw 相対的 表示領域の横幅（最小ビューポート基準）。
svh 相対的 表示領域の高さ（最小ビューポート基準）。
svmin 相対的 表示領域の横幅と高さの小さな方（最小ビューポート基準）
svmax 相対的 表示領域の横幅と高さの大きな方（最小ビューポート基準）
lvw 相対的 表示領域の横幅（最大ビューポート基準）。
lvh 相対的 表示領域の高さ（最大ビューポート基準）。
lvmin 相対的 表示領域の横幅と高さの小さな方（最大ビューポート基準）
lvmax 相対的 表示領域の横幅と高さの大きな方（最大ビューポート基準）
dvw 相対的 表示領域の横幅（ダイナミックビューポート基準）。
dvh 相対的 表示領域の高さ（ダイナミックビューポート基準）。
dvmin 相対的 表示領域の横幅と高さの小さな方（ダイナミックビューポート基準）
dvmax 相対的 表示領域の横幅と高さの大きな方（ダイナミックビューポート基準）

angle型（角度）

単位 説明
deg 角度を度数で表す。（全円は360deg）
grad 角度をグラジアンで表す。（全円は400grad）
rad 角度をラジアンで表す（全円は 2πラジアン（約6.2832rad））
turn 角度を回転数で表す。（全円は1turn）

resolution型（解像度）

単位 説明
dpi 1インチあたりのドット数。
dpcm 1cmあたりのドット数。
dppx 1pxあたりのドット数。
x ddpxのエイリアス。

time型

単位 説明
s 秒。
ms ミリ秒。（1/1000秒）。
