フレックスボックスとは

要素を行または列に一次元的に並べるレイアウト方法。

例えば、以下のような要素があるとする。

a
b
c
<style>
div.container-1_1 {
  border: 2px solid rgb(0, 0, 0);
  width: 100px;
  margin: 10px;

    .item {
      background-color: rgb(255, 201, 201);
      border: 2px dashed rgb(255, 0, 0);
      width: 50px;
      margin: 10px;
      text-align: center;
    }
  }
</style>

<div class="container-1_1">
  <div class="item">a</div>
  <div class="item">b</div>
  <div class="item">c</div>
</div>

フレックスボックス化したい要素の親要素にdisplay: flexを指定することで、親要素がフレックスコンテナ、子要素がフレックスアイテムとなる。

  <div class="item">a</div>

フレックスコンテナはボックスモデルに属するが、displayプロパティに指定する値に応じてブロックボックスかインラインボックスかを指定できる。

プロパティ/値 説明
display: flex 親要素がブロックボックスのフレックスコンテナを作成する。
display: inline-flex 親要素がインラインボックスのフレックスコンテナを作成する。

フレックスコンテナ/フレックスアイテム

フレックスボックス化した要素はフレックスコンテナとフレックスアイテムに区分される。

  • フレックスボックス化した親要素はフレックスコンテナとなる。
  • フレックスコンテナの子要素はフレックスアイテムとなる。
  • フレックスアイテムを並べる方向を主軸という。
  • 主軸に対して垂直の方向性を交差軸という。

フレックスアイテムの配置

列/行

flex-directionプロパティでフレックスアイテムの配置方向を指定できる。

プロパティ値 説明
row 左から右
row-reverse 右から左
column 上から下
column-reverse 下から上

Word に用意されているヘッダー、フッター、表紙、テキストボックスデザインを組み合わせると、プロのようなできばえの文書を作成できます。たとえば、一致する表紙、ヘッダー、サイドバーを追加できます。[挿入]をクリックしてから、それぞれのギャラリーで目的の要素を選んでください。テーマとスタイルを使って、文書全体の統一感を出すこともできます。[デザイン]をクリックし新しいテーマを選ぶと、図やグラフ、SmartArtグラフィックが新しいテーマに合わせて変わります。スタイルを適用すると、新しいテーマに適合するように見出しが変更されます。

一括指定

flex-flowプロパティでflex-directionとflex-wrapの値を一括指定できる。 

.container-1 {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
}

/* 上記と同等 */
.container-2 {
  flex-flow: row wrap;
}

フレックスアイテムのサイズ

アイテム毎のサイズ指定

flex-growプロパティでフレックスアイテムのサイズの伸縮を設定できる。

  • flex-growプロパティの値を指定するとフレックスアイテム内の余白部分を指定要素のサイズに割り当てる。
  • 複数のflex-growプロパティを指定した場合、アイテム単体のflex-grow指定値 / コンテナ内のflex-growプロパティの値の総数でサイズが計算される。
  • widthやheightよりも優先される。
  • 参考:【CSS】flex-growを使おう

以下は適用例。

item-1

item-2

item-3

<style>
  div.container-5_1 {
    display: flex;
    flex-direction: row;
    margin: 10px;
    gap: 10px;

    .item {
      background-color: rgb(168, 232, 227);
      border: 2px dashed rgba(155, 34, 132, 0.793);
      font-size: smaller;
    }

    .item-1 {
      /* 1/5(20%) */
      flex-grow: 1;
      order: 1;
    }

    .item-2 {
      /* 1/5(20%) */
      flex-grow: 1;
      order: 2;
    }

    .item-3 {
      /* 3/5(60%) */
      flex-grow: 3;
      order: 3;
    }
  }
</style>

<div class="container-5_1">
  <article class="item item-1">
    <p>item-1</p>
  </article>
  <article class="item item-2">
    <p>item-2</p>
  </article>
  <article class="item item-3">
    <p>item-3</p>
  </article>
</div>

アイテムのデフォルトサイズ指定

flex-basisプロパティでフレックスアイテムのデフォルトサイズを指定できる。
以下の値を指定可能。

プロパティ値 説明
auto サイズを自動で調整（規定値）
width型 widthやheightで指定可能な値

flex-growやflex-shrinkが指定されている場合はそちらの値も考慮される。
以下は適用例。

item-1

item-2

item-3


div.container-5_2 {
  display: flex;
  flex-direction: row;
  margin: 10px;
  gap: 10px;

  .item {
    background-color: rgb(168, 232, 227);
    border: 2px dashed rgba(155, 34, 132, 0.793);
    font-size: smaller;
  }

  .item-1 {
    /* 300px分のサイズ割り当て */
    flex-basis: 300px;
    order: 1;
  }

  .item-2 {
    /* 全体の20% */
    flex-basis: 20%;
    order: 2;
  }

  .item-3 {
    order: 3;
  }
}

サイズオーバー時の縮小値指定

flex-shrinkプロパティでフレックスアイテムのサイズの縮小値を指定できる。
全フレックスアイテムのサイズがフレックスコンテナのサイズを超過した際にflex-shrinkの値が適用される。

一括指定

flexプロパティで一括指定できる。

.container {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;

  .item-1 {
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: auto;
  }

  /* .item-1と同等 */
  .item-2 {
    flex: 1 1 auto;
  }
}

主軸方向

justify-contentプロパティで主軸方向の全体的なレイアウトを指定できる。

図で理解する Flexboxより引用
プロパティ値 説明
flex-start 先頭寄せ
flex-end 末尾寄せ
center 中央寄せ
space-between
  • 均等配置
  • コンテナとアイテム間の余白なし
space-around
  • 均等配置
  • コンテナとアイテム間の余白は、隣接するアイテム間の余白の半分になる
space-evenly
  • 均等配置
  • コンテナとアイテム間の余白は、隣接するアイテム間の余白と同じになる

交差軸方向（全体）

align-content プロパティで交差軸方向の全体的なレイアウトを指定できる。
以下の値を指定できる。

図で理解する Flexboxより引用
プロパティ値 説明
flex-start 先頭寄せ
flex-end 末尾寄せ
center 中央寄せ
space-between
  • 均等配置
  • コンテナとアイテム間の余白なし
space-around
  • 均等配置
  • コンテナとアイテム間の余白は、隣接するアイテム間の余白の半分になる
space-evenly
  • 均等配置
  • コンテナとアイテム間の余白は、隣接するアイテム間の余白と同じになる

交差軸方向（行）

align-items プロパティで各フレックスアイテム行の交差軸方向のレイアウトを指定できる。
以下の値を指定できる。

図で理解する Flexboxより引用
プロパティ値 説明
flex-start 先頭寄せ
flex-end 末尾寄せ
center 中央寄せ
baseline 各アイテムのベースライン（テキストの位置）が同じになるよう配置
stretch 各アイテムのサイズが同じになるよう配置

交差軸方向（単体）

align-self プロパティで各フレックスアイテム単体の交差軸方向のレイアウトを指定できる。
以下の値を指定できる。

図で理解する Flexboxより引用
プロパティ値 説明
flex-start 先頭寄せ
flex-end 末尾寄せ
center 中央寄せ
stretch 各アイテムのサイズが同じになるよう配置

一括指定

place-contentプロパティでjustify-contentとalign-contentを一括指定できる。 

.container {
  display: flex;

  .item-1 {
    align-content: space-evenly;
    justify-content: space-between; 
  }

  /* .item-1と同等 */
  .item-2 {
    place-content: space-evenly space-between;
  }
}
