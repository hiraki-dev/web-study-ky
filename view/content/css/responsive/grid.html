グリッドレイアウトとは

Webページの表示領域を格子状のマス目に見立てたとき、どのマス目にどの要素を配置するかを指定するレイアウト方法。
グリッドは列、行、行列間の間隔（溝/ガター）で構成されている。

グリッドコンテナ/グリッドアイテム

グリッドはグリッドコンテナとグリッドアイテムで構成される。

例えば、以下のような要素があるとする。

aaa
bbb
ccc
ddd
eee

ggg

グリッド化したい要素の親要素にdisplay: gridを指定することで、親要素がグリッドコンテナ、子要素がグリッドアイテムとなる。

bbb
ccc
ddd

ggg
<style>
  .container-2 {
    display: grid;
    grid-template-columns: 100px 100px 100px;
    gap: 10px;

    & .item {
      margin: 10px;
      border: 1px solid black;
      border-radius: 4px;
      width: 100px;
    }
  }
</style>

  <div class="item">ccc</div>

  <div class="item">fff</div>

グリッドコンテナはボックスモデルに属するが、displayプロパティに指定する値に応じてブロックボックスかインラインボックスかを指定できる。

プロパティ/値 説明
display: grid 親要素がブロックボックスのグリッドコンテナを作成する。
display: inline-grid 親要素がインラインボックスのグリッドコンテナを作成する。

グリッドライン/グリッドトラック

グリッドの縦横の区切り線のことをグリッドラインという。
また、グリッドライン間の空間のことをグリッドトラックといい、グリッドトラック上にグリッドアイテムを配置する。

  • 行方向のグリッドトラックは行トラックという。
  • 列方向のグリッドトラックは列トラックという。

明示的なグリッドトラック/暗黙的なグリッドトラック

グリッドトラックには明示的なグリッドトラックと暗黙的なグリッドトラックの二種類がある。
以下のプロパティで明示的に行トラック/列トラックのサイズを指定した場合、明示的なグリッドトラックとなる。

  • grid-template-rows
  • grid-template-columns
  • grid-template-areas

明示的なグリッドトラックが指定されておらず、グリッドアイテムの数がグリッドトラックの数を超過した場合に暗黙的にグリッドトラックが自動的に拡張される。
これを暗黙的なグリッドトラックといい、以下のプロパティでサイズを指定する。

  • grid-auto-rows
  • grid-auto-columns

グリッドライン番号

グリッドラインは行方向、列方向でラインの数に応じて番号が割り振りされている。

グリッドセル/グリッドアイテム

グリッドアイテムを配置する際、グリッドのマス目1マスの単位をグリッドセルという。
グリッドトラック上の複数のグリッドセル領域のことをグリッドエリアという。

  • グリッドエリアはグリッドライン上に沿うように定義される。
  • そのため、グリッドエリアは必ず長方形の形となる。

グリッドセルの割合指定

fr単位を用いることでグリッドトラックの各セル領域の割合を指定出来る。
各セルの割合は各グリッドセルのfr値/トラック上のfr値の合計で算出される。 

<style>
  .container-4_1 {
    display: grid;
    grid-template-columns: 2fr 1fr 1fr;
    gap: 10px;

    .item {
      margin: 10px;
      border: 1px solid black;
      border-radius: 4px;
      text-align: center;
    }
  }
</style>

<div class="container-4_1">
  <div class="item">グリッドセル<br />2つ分</div>
  <div class="item">グリッドセル<br />1つ分</div>
  <div class="item">グリッドセル<br />1つ分</div>
  <div class="item">グリッドセル<br />2つ分</div>
</div>

グリッドガターの指定

グリッドセル間の間隔をグリッドガターという。
以下のプロパティでグリッドガターを指定できる。

プロパティ 説明
row-gap 行間の間隔
column-gap 列間の間隔
gap 行列の間隔を一括指定

仕組みとしては、repeat関数で指定サイズ領域を反復させてグリッドトラック領域のセルを定義する。

グリッドセルの最大値/最小値指定

minmax関数を用いることで各グリッドセルの最小値と最大値を指定出来る。
単一のセルに指定出来るほか、repeat関数との組み合わせでグリッドトラックのセルの柔軟な指定も可能。

  • 第一引数に最小値を指定する。
  • 第二引数に最大値を指定する。
    • autoを指定するとブラウザに合わせて自動でサイズを算出する。

グリッドトラック配置

grid-template-rows、grid-template-columnsプロパティでグリッドトラック領域のサイズを指定出来る。

行トラック指定

行トラックはgrid-template-rowsプロパティで指定する。

.container {
  display: grid;
  grid-template-rows: 1fr, 2fr, 1fr;
}

列トラック指定

列トラックはgrid-template-columnsプロパティで指定する。

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

暗黙的グリッドトラックの明示指定

暗黙的なグリッドトラックはデフォルトではautoでサイズ指定される。
grid-auto-rowsプロパティとgrid-auto-columnsプロパティを指定することで暗黙的グリッドトラックのサイズを明示して指定出来る。 

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 1fr 2fr 1fr;
}

インライン軸/ボックス軸配置

グリッドアイテムを並べる場合、行方向はインライン軸、列方向はブロック軸という。

インライン軸（全体）

justify-contentプロパティでインライン軸方向の全体的なレイアウトを指定できる。
以下の値を指定できる。

図で理解する Grid Layoutより引用
プロパティ値 説明
start 先頭寄せ
end 末尾寄せ
center 中央寄せ
space-between
  • 均等配置
  • コンテナとアイテム間の余白なし
space-around
  • 均等配置
  • コンテナとアイテム間の余白は、隣接するアイテム間の余白の半分になる
space-evenly
  • 均等配置
  • コンテナとアイテム間の余白は、隣接するアイテム間の余白と同じになる

stretch 各アイテムのサイズを均一にして配置

justify-selfプロパティでインライン軸方向のアイテム単体のレイアウトを指定できる。
以下の値を指定できる。

一括指定

以下のプロパティでボックス軸とインライン軸のレイアウトを一括指定出来る。

  • place-content
    • justify-contentとalign-contentを一括指定できる。
    • 1つ目の値がalign-contentを指定する。
    • 2つ目の値はjustify-contentを指定する。
  • place-items
    • justify-itemsとalign-itemsを一括指定できる。
    • 1つ目の値がalign-itemsを指定する。
    • 2つ目の値はjustify-itemsを指定する。
  • place-self
    • justify-selfとalign-selfを一括指定できる。
    • 1つ目の値がalign-selfを指定する。
    • 2つ目の値はjustify-selfを指定する。
.container {
  display: grid;
  place-content: start end;
  
  & .item {
    place-items: start end;
    place-self: start end;
  }  
}

名前付きグリッドエリア

grid-template-areasプロパティでグリッドエリアに名前を定義することが出来る。
grid-areaプロパティを用いてグリッドアイテムに定義した名前を指定することで任意のエリアにアイテムを配置できる。

  • grid-template-areas: エリア名 エリア名 ...と記述して名前付きグリッドエリアを定義する。
    • エリア名はスペースで区切る
    • エリア名は重複して記載でき、その場合複数のグリッドセルを使用したエリアとなる
    • スペースで区切った数が列トラックのアイテム数となる
    • 改行してエリア名を記述することで行トラックを表現できる
    • 何も指定しないエリア（無名グリッドエリア）はnone（または.）を指定する
  • グリッドアイテムにgrid-area: エリア名と指定することでエリア名を記述した位置にアイテムを配置できる。
  • grid-template-rows、grid-template-columnsで名前付きグリッドエリアの領域幅の指定が出来る。

以下は定義例。

header
main
footer
side
aside
<style>
  .container-8 {
    display: grid;
    grid-template-areas:
      "area-header area-header"
      "area-main area-side"
      "area-main area-aside"
      "area-main ."
      "area-footer area-footer";
    grid-template-rows: 50px 150px 100px 50px 50px;
    /* grid-template-rows: 1.25fr 3.75fr 2.5fr 1.25fr 1.25fr; */
    grid-template-columns: 2fr 1fr;
    gap: 10px;
  
    & .header {
      grid-area: area-header;
      background-color: red;
    }
  
    & .main {
      grid-area: area-main;
      background-color: yellow;
    }
  
    & .footer {
      grid-area: area-footer;
      background-color: blue;
    }
  
    & .side {
      grid-area: area-side;
      background-color: green;
    }
  
    & .aside {
      grid-area: area-aside;
      background-color: orange;
    }
  }
</style>
  
<div class="container-8">
  <div class="header">header</div>
  <div class="main">main</div>
  <div class="footer">footer</div>
  <div class="side">side</div>
  <div class="aside">aside</div>
</div>
