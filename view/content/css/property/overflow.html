概要

親要素内の子要素が親要素のサイズを超過した場合、親要素からはみ出て表示される。
これをオーバーフローといい、値の消失を防ぐために意図的に表示される設計になっている。

ビデオを使うと、伝えたい内容を明確に表現できます。[オンラインビデオ]をクリックすると、追加したいビデオを、それに応じた埋め込みコードの形式で貼り付けできるようになります。キーワードを入力して、文書に最適なビデオをオンラインで検索することもできます。Word に用意されているヘッダー、フッター、表紙、テキスト ボックスデザインを組み合わせると、プロのようなできばえの文書を作成できます。たとえば、一致する表紙、ヘッダー、サイドバーを追加できます。[挿入]をクリックしてから、それぞれのギャラリーで目的の要素を選んでください。テーマとスタイルを使って、文書全体の統一感を出すこともできます。[デザイン]をクリックし新しいテーマを選ぶと、図やグラフ、SmartArtグラフィックが新しいテーマに合わせて変わります。スタイルを適用すると、新しいテーマに適合するように見出しが変更されます。

overflowの種類

ここでは、overflowプロパティの種類と値について纏めておく。

selector {
  /* 横軸のオーバーフローの挙動を指定 */
  overflow-x: 値;
  /* 縦軸のオーバーフローの挙動を指定 */
  overflow-y: 値;
  /* 縦軸横軸のオーバーフローの挙動を指定 */
  overflow: 値;
  overflow: x軸の値 y軸の値;
}
プロパティ値 種類
visible オーバーフロー時に要素を表示する。
hidden オーバーフロー時に要素を表示しない。
auto オーバーフロー時にスクロールバーを表示する。
scroll オーバーフローの有無に関わらずスクロールバーを表示する。
