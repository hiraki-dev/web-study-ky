概要

animationプロパティを用いてCSSレイアウトにアニメーションを付与することができる。
以下はCSSアニメーションの例である。 

<style>
  .animation_sample {
    overflow: hidden;

    & img {
      width: 25%;

      /* アニメーション名を定義 */
      animation-name: chikawa-slide;
      /* アニメーションの実行時間を定義 */
      animation-duration: 8s;
    }
  }

  /* アニメーションの挙動を定義 */
  @keyframes chikawa-slide {
    /* 開始時のスタイル */
    from {
      margin-left: 100%;
    }

    /* 終了時のスタイル */
    to {
      margin-left: 0%;
    }
  }
</style>

  • animation関連プロパティでアニメーションの定義、実行時間、動作の繰り返しなどを定義する。
  • @keyframesルールでアニメーションの動きを定義する。

animationプロパティ

animationプロパティには以下の種類がある。

プロパティ 説明
animation-name @keyframesで定義したフレーム名を指定してセレクタとアニメーションの紐付けを行う。
animation-duration アニメーションの進行が完了するまでの時間を指定する。
animation-delay アニメーションを開始するまでの遅延時間を指定する。
animation-iteration-count アニメーションの繰り返し回数を指定する。
規定値は1で、infiniteを指定すると無限ループされる。
animation-play-state アニメーションの実行/一時停止を指定する。JavaScriptなどと併用して使用する。
  • runningでアニメーションを一時停止する。
  • pausedでアニメーションを一時停止する。
animation-timing-function アニメーション進行がどのように行われるかを指定する。
animation-direction アニメーションの実施方向を指定する。
animation-fill-mode アニメーションの開始/終了時のスタイルを指定する。
animation 上記のanimationプロパティを一括定義する

以下はanimationプロパティの指定例。

<style>
  .animation_property_sample {
    width: 100vw;
    height: 130px;

  & img {
    width: 20%;

      animation-name: chikawa-slide-custum;
      animation-duration: 1s;
      animation-iteration-count: infinite;
      animation-timing-function: linear;
      animation-direction: alternate;
      animation-delay: normal;
      animation-fill-mode: both;
    }
  }

  @keyframes chikawa-slide-custum {
    0% {
      opacity: 100%;
    }

    20% {
      opacity: 70%;
      scale: 80%;
    }

    35% {
      opacity: 50%;
    }

    50% {
      opacity: 25%;
      scale: 75%;
    }
  
    70% {
      opacity: 50%;
    }

    85% {
      opacity: 70%;
      scale: 110%;
    }

    100% {
      opacity: 100%;
    }
  }
</style>

animation-timing-function

アニメーションがどのように進行されるかを指定する。
以下の値がある。

プロパティ値 説明
linear 均等に変化する。
ease 中央に向けて素早く変化した後、ゆっくり終了していく。（規定値）
ease-in 中央に向けてゆっくり変化した後、素早く終了していく。
ease-out easeと同様だが、開始スピードがより早い。
ease-in-out 最初と最後がゆっくりで、中央に近づくほど速くなる。
step-start アニメーション開始時の状態で固定する。
step-end アニメーション終了時の状態で固定する。
steps(ステップ数, start|end) ステップ数に指定した値分コマ送りする。
第二引数にstartを指定するとすぐにアニメーションが開始され、endを指定するとインターバルを置いてから実行する。
cubic-bezier(x1, y1, x2, y2) アニメーション進行を任意にカスタムする。
x軸は時間経過率で0.0 ~ 1.0の値、y軸は変化率を任意に指定する。
以下は参考値である。
  • innear: cubic-bezier(0.0, 0.0, 1.0, 1.0)
  • ease: cubic-bezier(0.25, 0.1, 0.25, 1.0)

以下はanimation-timing-function: ease-in-out;を指定した例。

animation-direction

アニメーションの実施方向を指定する。

プロパティ値 説明
normal 「開始から終了」と進行。（規定値）
reverse 「終了から開始」と進行。
alternate 「開始から終了」の後、「終了から開始」と進行。
alternate-reverse 「終了から開始」の後、「開始から終了」と進行。

animation-fill-mode

アニメーション開始時、終了時のスタイルを制御する。

プロパティ値 説明
none 開始/終了時にアニメーションを適用せず、ワンテンポ置いて実行する。（規定値）
both 開始/終了時にアニメーションを適用する。
forwards アニメーション終了時、終了時のスタイルで固定する。
backwards アニメーション開始時のスタイルを適用する。
また、animation-delay指定時、アニメーション開始時のスタイルを適用する。

@keyframesルール

アニメーションのフレームレートを定義する。
定義したフレームレート毎にスタイルを適用できる。

  • 開始を0%、終了を100%として定義する。
  • 0%と100%のみを定義する場合、from、toで表記できる。
  • @keyframesルールはネストの中に定義することはできない。
.selector {
  animation-name: test;
  animation-duration: 3s;
}

@keyframes test {
  0% {
    ...
  }

  50% {
    ...
  }

  100% {
    ...
  }
}
