フォントファミリーの指定（font-family）

Webページのフォントファミリーを指定する。
以下はフォントファミリーの指定例である。 

    * {
  font-family: "ヒラギノ角ゴ Pro W3", "Hiragino Kaku Gothic Pro", "メイリオ", "Meiryo", "ＭＳ Pゴシック", "MS PGothic", sans-serif;
}
    • プロパティ値に指定するフォント名称をフォントファミリー名といい、フォントファミリー名を指定することでWebページのテキストが指定フォントで反映される。
    • 複数指定する場合はカンマ区切りで指定を行い、指定順（右から左）に優先度が適用される。
    • フォントは1文字ずつに適用される、指定フォントが適用できない場合は優先度順に指定したフォントが反映される。

    総称フォントファミリー

    総称フォントファミリーを指定することで、指定フォントが全て適用できない場合に適用するシステムフォントを選択出来る。
    主に以下のプロパティ値がサポートされており、指定した値に応じたシステムフォントが適用される。

    • 総称フォントファミリーを指定する場合、フォントリストの最後である必要がある。
    • 総称フォントファミリーは引用符で囲わずに指定する。
    プロパティ値 字体 適用フォントファミリー名
    serif 明朝体 MS P明朝, ヒラギノ明朝 等
    sans-serif ゴシック体 メイリオ, ヒラギノ角ゴシック, Arial 等
    monospace 等幅フォント Sans Mono, Menlo 等
    cursive 筆記体 参考
    fantasy 装飾フォント 参考
    system-ui OS規定のフォント OSに依存

    フォントファイルの指定（@font-face）

    @font-faceルールを用いることでフォントファイルを指定できる。
    以下は指定例である。 

    /* フォントファイルの定義 */
@font-face {
  /* フォントファミリー名を定義 */
  font-family: "custum font",
  /*
    フォントパスとフォント形式を指定
    指定可能なフォント形式は以下
    "woff",
    "woff2",
    "truetype",
    "opentype",
    "embedded-opentype",
    "svg"
  */
  src: url(../fonts/xxx.otf) format("opentype");

  /* デフォルト値の指定も可能 */
  font-size: 24px;
  font-weight: bold;
}

a {
  /* フォントファイルの反映 */
  font-family: "custum font";
}
