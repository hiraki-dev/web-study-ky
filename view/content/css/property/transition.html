概要

transitionプロパティを用いることで、CSSスタイルが適用されるまでの時間を指定できる。
transitionプロパティは以下の種類がある。

プロパティ 説明
transition-property トランジションを適用するプロパティを指定する。
tansition-duration トランジションの完了時間を指定する。
tansition-delay トランジション開始までの遅延時間を指定する。
tansition-timing-function トランジションの進行がどのように行われるかを指定する。
プロパティ値はanimation-timing-functionと同様である。
transition 上記のanimationプロパティを一括定義する。

以下はtransitionの例である。

<style>
  .transition_sample {
    overflow: hidden;

    & img {
      width: 200px;

      transition-property: all;
      transition-duration: 0.5s;
      transition-delay: 0.2s;
      transition-timing-function: ease-in-out;
    }

    & img:hover {
      border-radius: 50%;
      rotate: 360deg;
      background-color: rgba(255, 255, 1, 0.5);
    }
  }
</style>

<div class="transition_sample">
  <img
    src="../../../asset/images/chikawa_1.png"
    alt="トランジションの例"
  />
</div>

また、以下の点を留意して使用すること。

  • トランジションはCSS内部で自動計算する都合上、autoプロパティを使用すると内部計算が非常に複雑になりバグの原因になる。
    そのためトランジションとautoプロパティの併用は避ける。
  • 類似するanimationプロパティとは適用タイミングが異なる。
    animationプロパティはページがビューポートに読み込まれたタイミングで適用されるが、transitionプロパティは疑似クラスやJavaScriptのイベントなどが発火されたタイミングで適用される。

transition-property

トランジションを適用するプロパティを指定する。

プロパティ値 説明
all 全プロパティに適用する。（規定値）
none 全プロパティに適用しない。

プロパティを複数選択する場合、以下のようにカンマ区切りで指定する。

.selector {
  transition-property: width, height;
}

tansition-timing-function

トランジションがどのように進行されるかを指定する。

プロパティ値 説明
linear 均等に変化する。
ease 中央に向けて素早く変化した後、ゆっくり終了していく。（規定値）
ease-in 中央に向けてゆっくり変化した後、素早く終了していく。
ease-out easeと同様だが、開始スピードがより早い。
ease-in-out 最初と最後がゆっくりで、中央に近づくほど速くなる。
step-start トランジション開始時の状態で固定する。
step-end トランジション終了時の状態で固定する。
steps(ステップ数, start|end) ステップ数に指定した値分コマ送りする。
第二引数にstartを指定するとすぐにトランジションが開始され、endを指定するとインターバルを置いてから実行する。
cubic-bezier(x1, y1, x2, y2) トランジション進行を任意にカスタムする。
x軸は時間経過率で0.0 ~ 1.0の値、y軸は変化率を任意に指定する。
以下は参考値である。
  • innear: cubic-bezier(0.0, 0.0, 1.0, 1.0)
  • ease: cubic-bezier(0.25, 0.1, 0.25, 1.0)
